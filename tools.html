Tools & Utilities


For most real-life application you would need some utilities alongside callbag-jsx. Here are a list of libraries for most common use cases such as state management, routing and testing.

linkState Management

Because callbag-jsx is designed around seamless embedding of callabgs in JSX, the most convenient tools for data-flow management are naturally those that are callbag based. Specifically the following libraries are recommended:

  • ‚≠ê callbag-common: Commonly used utilities for working with callbags
  • ‚≠ê callbag-state: State management with callbags

Alternatively, you could use any of the following libraries:

  • callbag-basics: Basic callbag utilities
  • callbag-subject: A listener sink that is also a listenable source, like RxJS Subject
  • callbag-behavior-subject: A subject that repeats its last value on connection (like RxJS Behavior Subject)
  • reduxbag: Redux middleware for callbags

üëâ Checkout callbag wiki for a list of callbag-based tools for state and data-flow management.
